# IG_Img_Object_Detection

- 利用爬蟲selenium開啟Instagram,透過Hashtag搜尋關鍵字,將關鍵字的圖片下載下來
- 使用Tkinter製作GUI介面,並進行YOLOv7的辨識
- 目前辨識類別只有**貓**.**狗**.**馬**.**鳥**
